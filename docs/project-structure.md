#1 Project Structure Overview
This document gives an overview of the project structure of next.js+django devops deployment project

#2 frontend --> Next.js code
This directory contains Next.js application source code.

#3 backend --> Django code
This directory contains Django Rest API source code.

#4 Docker
This directory contains dockerfiles and docker-compose configurations for containerization.

#5 terraform
This directory contains Infrastructure as Code (IaC) files used to provision AWS resources.

#6 nginx
This directory contains reverse proxy configuration that helps routing traffic to frontend and backend services.

#7 docs
This directory contains all the project documentation, architecture diagrams, screenshots, and implementation notes.

#8 monitoring
This directory holds prometheus and grafana configuration for observability and metrics.

#10 scripts
This directory holds scripts for backup and restore utilities.
